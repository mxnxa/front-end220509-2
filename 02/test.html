HTML + CSS

  • HTML이란 무엇인지 배워보고, HTML의 기초 문법과 기본 구조를 이해하고 작성할 수 있다.

  • CSS의 역할이 무엇인지 알아보고, CSS의 기본 문법과 여러 레이아웃 기법을 익히고 활용할 수 있다.

  • HTML로 기본적인 웹 문서 내용을 작성하고,
    CSS로 HTML 문서를 레이아웃과 디자인을 사용자가 보기 편하게 변경할 수 있다.
