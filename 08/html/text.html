font-family

font-family는 글꼴을 지정하는 css 속성이다.

font-size

font-size는 크기를 지정하는 속성이다.

font-style

font-style은 글자를 이탤릭체로 바꾸는 속성이다.

color

color는 글씨의 색상을 바꿔주는 속성이다.

text-align

text-align은 글 정렬을 지정하는 속성이다.

line-height

line-height는 줄간격을 지정하는 속성이다.

text-decoration

text-decoration은 텍스트에 밑줄을 긋거나 취소선을 표시 한다.

text-transform

TEXT-transform 텍스트를 대소문자로 변경한다.

text-shadow

text-shadow는 글자에 그림자를 넣는다.

letter-spacing, word-spacing

letter-spacing은 자간을, word-spacing은 어간을 조절한다.
